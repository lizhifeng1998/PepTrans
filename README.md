# PepTrans

Transformer-based model for peptide stability prediction. Encodes amino acid sequences with learnable embeddings fused with physicochemical features, then regresses a stability score via a Transformer encoder.

## Installation

Follow the [PepFlow](https://gitlab.com/oabdin/pepflow) environment setup, then install the additional dependency:

```bash
pip install scikit-learn
```

## Data Format

Input data files (e.g. `stabil045.txt`) are whitespace-delimited plain text:

```
AAAWC 0.04
AAYRL 0.34
AARFDGKFY 0.81 0.02
```

| Column | Description |
|--------|-------------|
| 1 | Amino acid sequence (uppercase, variable length) |
| 2 | Stability value (0.01–1.00) |
| 3 | *(optional)* Stability value of the secondary structure |

The file is organized in two sections:

- **Rows 1–32,000**: Initial random sampling data.
- **Rows 32,001+**: iterative batches generated by active learning. Batch boundaries are determined by alphabetical reset of the first residue — when the leading amino acid of a row precedes (in A→Y order) that of the previous row, a new batch begins.

## Usage

Four modes are available via subcommands. Run `python trans.py <mode> -h` for full options.

### Train

```bash
python trans.py train --data stabil045.txt --epochs 150 --lr 1e-4
```

Key options: `--model-path`, `--batch-size`, `--weight-decay`, `--no-overwrite`.

### Validate

```bash
python trans.py validate --data stabil045.txt --model-path best_peptide_model.pth
```

Use `--threshold 0.8` to evaluate only on samples with stability > 0.8.

### Predict

```bash
# Predict specific sequences
python trans.py predict --model-path best_peptide_model.pth --sequences AGVKL LIPKW

# Predict from a file (one sequence per line)
python trans.py predict --model-path best_peptide_model.pth --seq-file seqs.txt

# Filter from data file (default: held-out sequences with value > -0.8)
python trans.py predict --data stabil045.txt --filter-threshold 0.5 --output preds.txt
```

### Extrapolate

Randomly generate sequences and screen for high-stability candidates:

```bash
python trans.py extrapolate --threshold 0.9 --seq-len-min 8 --seq-len-max 14 \
    --target-count 100 --output candidates.txt
```
